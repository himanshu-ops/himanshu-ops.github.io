1. Hospital charges prediction

This project contains a flask API based prediction model based on Linear regression that predicts the total charges of an inpatient discharge based on historical data and factors that influence this cost. This project is designed by keeping end user in mind and provides an intuitive and simple user interface.

According to statista.com the per capita healthcare expenditure for the United States is highest in the world. This number is almost double than the country with the second highest per capital expenditure which is Germany. This is a cause for concern for the population that come under low-income group and the population that do not have or cannot afford health insurance. This project tries to solve this problem and focuses on creating a user-friendly web-based application that can predict the total charges for inpatient discharge based on various factors. The application is based on a machine learning model which Is trained on historical data from the year 2015 for the total charges for inpatient discharge.

Result: The model achieved an accuracy of 92%. Also from the plots we found that majority of the patients are from the category 70 and above followed by 50 to 69 years.
